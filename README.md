# 🧠 Rare OS

A personal operating system dashboard built as a Progressive Web App (PWA). Manage your projects, tasks, devices, and more with a sleek, modern interface.

## ✨ Features

- 📱 **Progressive Web App** - Install as a native app on iOS and Android
- 📴 **Offline Support** - Full functionality without internet connection
- 🎨 **Modern UI** - Beautiful, responsive design optimized for all devices
- 👤 **User Profiles** - Create and customize your personal profile
- 📁 **Project Management** - Track and manage your projects
- ✅ **Task Management** - Keep track of your daily tasks
- 📱 **Device Management** - Monitor connected devices
- 📝 **Quick Notes** - Capture ideas and notes instantly
- 📊 **Dashboard** - Overview of your health, printer, car, and more
- 🔔 **Notifications** - Stay updated with browser notifications
- 📱 **Timeline** - View your activity history

## 🚀 Getting Started

### Installation

1. Visit the GitHub Pages URL for your repository
2. On mobile devices, tap "Share" and select "Add to Home Screen"
3. On desktop, click the install icon in your browser's address bar

### First Time Setup

1. When you first open the app, a profile creation dialog will appear
2. Enter your name, occupation, and bio
3. Click "Save Profile" to continue
4. Your profile will be saved locally on your device

## 📱 How to Use

### Quick Actions

- **📁 Project** - Create a new project
- **✅ Task** - Add a new task to your daily list
- **📝 Note** - Capture a quick note or idea
- **📱 Device** - Add a new device to your network

### Dashboard Widgets

- **❤️ Health** - System health status
- **🖨️ Printer** - Printer connectivity status
- **🚗 Car** - Vehicle status
- **📅 Today** - Today's tasks count
- **🤖 AI** - AI Assistant (Coming Soon)
- **💻 Devices** - Number of connected devices

### Notifications

Tap the 🔔 button to enable push notifications and stay updated with your app's activities.

## 💾 Data Storage

All your data is stored locally in your browser using localStorage. Your data is private and never sent to external servers.

## 🔒 Privacy

Your personal data is stored entirely on your device. We don't collect, store, or transmit any of your information to external servers.

## ⚙️ Technical Stack

- **Frontend:** HTML5, CSS3, Vanilla JavaScript
- **Storage:** LocalStorage API
- **Caching:** Service Workers
- **Framework:** Progressive Web App (PWA)
